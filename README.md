# UnbiasedAIBenchmark
Unbiased AI BenchMarks

### Why
Benchmarks i see around never match with my personal experience using the models. I want to offer a real unbiased tests everyone can run.

### Models tested

Models tested are only free versions available for everyone to use.  

  
- GPT-5.2 Instant 
- Gemini 3.0 Flash
- DeepSeek 3.2
- Sonnet 4.5
- Kimi K2


### About tests

Tests will be listed below with the results from each model and a score from 0 to 100 based on my analisis of the response. Same prompt used to all models.
